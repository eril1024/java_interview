1.HelloWorld 를 출력하는 프로그램의 과정을 설명하시오.
사용자가 HelloWorld를 출력하는 명령어를 입력하고 컴파일하면은 javac가 사용자가 작성한 코드를 컴파일한다.이상이 없을시 jvm이 작동하며
컴파일한 결과를 출력해준다. 

2.아래의 명령어를 설명하시오(뭐하는건가).
-javac
자바 컴파일러로 우리가 자바언어로 작성한 소스코드를 컴퓨터언어인 이진수로 변환 하는 것이다.
-java
우리가 작성한 소스코드를 구동할 수 있게 해주는 프로그램을 작동시키는 명령어이다.

3.컴파일이란 무엇인가요?
사용자와 컴퓨터간에 의사소통이 가능하게 번역해주는 프로그램으로 사용자가 작성한 소스코드를 컴퓨터가 읽을 수 있게 컴파일러가
컴퓨터언어로 번역 해준다.

4.java 언어를 창시한 사람은?
제임스 고슬링

5.JDK란 무엇이며, 어디서 다운로드 받으며, OS별로 버전이 있는 까닭은?
jdk란 자바 개발자들이 프로그램을 만들 수 있도록 지원해주는 개발 키트를 말한다. 오라클사이트에서 받을 수 있으며 각 OS마다 실행환경
이 다르기 때문에 자신의 컴퓨터에 맞는 운영체제 버전을 다운받아 사용하면된다.

6.JVM 무엇의 약자이며,이란 무엇인가?
java virtual machine의 약자로 class로 컴파일된 파일을 구동시켜주는 프로그램이다.
